## How to build (ubuntu)
* install dependencies ;

```shell
	sudo apt install cmake cmake-curses-gui python perl pkg-config bison flex gperf
	sudo apt install libxml2-dev libcairo2-dev libsdl1.2-dev
```

* how to build ;

```shell
	mkdir ../build-axSurf
	cd ../build-axSurf
	cmake ../axSurface/trunk
```
